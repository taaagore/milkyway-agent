# FSSAI Grounding Skill

This skill handles the grounding and verification of lactometer readings against official FSSAI standards.

## Usage

Use this skill to determine whether a lactometer reading is within the standard limits set by FSSAI for various types of milk (e.g., cow, buffalo, mixed, standardized).
